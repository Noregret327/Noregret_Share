## 绘图

### 折线图

```python
plt.plot(df["物品编号"],df["库存量"],label='库存量',linewidth=1,color='c',marker='o',markerfacecolor='blue',markersize=5)
#横坐标为物品编号，纵坐标为库存量，线的名称为库存量，粗细为1，颜色为青色，标记为“o”所代表的图形（会在后面详细介绍），颜色为蓝色，大小为5
```
